# 发现与决策

## 需求
- 独特风格，区别于普通博客
- 炫酷视觉效果
- 丰富图片和视觉元素
- 丰富功能
- 部署到GitHub

## 研究发现
- 当前使用 Hexo 8.1.1 + NexT 8.26.0 主题 (Gemini方案)
- 主题从npm安装，themes/目录为空
- 现有2篇文章，about页面，基本配置
- 已配置GitHub Pages部署到 yucu666/yucu666.github.io
- Hexo 支持自定义主题放在 themes/ 目录

## 技术决策
| 决策 | 理由 |
|------|------|
| 自定义主题 (neon-vault) | 完全掌控设计，不必受限于NexT结构 |
| CSS原生 + CSS变量 | 无编译依赖，浏览器原生支持 |
| EJS模板引擎 | Hexo默认支持，无需额外配置 |
| 霓虹赛博朋克美学 | 深色背景+霓虹发光+玻璃态，技术博客的完美匹配 |
| SVG内联插画 | 可缩放、可动画、文件小 |

## 设计方向
- **风格**：Cyberpunk / Neon Noir / Synthwave
- **主色调**：深紫(#1a0033) → 深蓝(#0a0a2e) 渐变背景
- **强调色**：霓虹粉(#ff2d95)、电光蓝(#00d4ff)、霓虹绿(#39ff14)
- **字体**：JetBrains Mono (代码) + 思源黑体 (正文)
- **效果**：玻璃态卡片、发光边框、粒子背景、扫描线、霓虹光晕

## 资源
- Google Fonts: JetBrains Mono, Noto Sans SC
- particles.js 用于粒子背景
- SVG 自制插画

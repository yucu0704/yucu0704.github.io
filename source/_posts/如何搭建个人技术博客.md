# 如何搭建个人技术博客

本文将介绍如何使用Hexo和GitHub Pages搭建一个免费的个人技术博客。

## 什么是Hexo？

Hexo是一个快速、简洁且高效的博客框架。它使用Markdown（或其他渲染引擎）解析文章，在几秒内即可生成静态网页。

## 准备工作

在开始之前，您需要安装以下软件：

1. [Node.js](https://nodejs.org/) (建议使用LTS版本)
2. [Git](https://git-scm.com/)
3. [GitHub账号](https://github.com/)

## 安装Hexo

打开终端或命令提示符，执行以下命令安装Hexo：

```bash
npm install -g hexo-cli
```

## 创建博客项目

1. 创建一个新的目录用于存放博客文件：

```bash
hexo init my-blog
cd my-blog
npm install
```

2. 本地预览博客：

```bash
hexo server
```

访问 http://localhost:4000 即可查看您的博客。

## 配置博客

编辑 `_config.yml` 文件，配置您的基本信息：

```yaml
# Site
title: 您的博客标题
subtitle: 副标题
description: 博客描述
author: 您的名字
language: zh-CN
```

## 选择主题

Hexo有许多优秀的主题可供选择。本文使用的是Next主题：

1. 安装Next主题：

```bash
npm install hexo-theme-next --save
```

2. 在 `_config.yml` 中设置主题：

```yaml
theme: next
```

3. 创建Next主题配置文件 `_config.next.yml`，详细配置主题样式。

## 部署到GitHub Pages

1. 在GitHub上创建一个新的仓库，仓库名称为 `your-username.github.io`。

2. 安装部署插件：

```bash
npm install hexo-deployer-git --save
```

3. 在 `_config.yml` 中配置部署信息：

```yaml
deploy:
  type: git
  repository: https://github.com/your-username/your-username.github.io.git
  branch: main
```

4. 生成并部署博客：

```bash
hexo clean
hexo generate
hexo deploy
```

## 写作与发布

1. 创建新文章：

```bash
hexo new "文章标题"
```

2. 编辑生成的Markdown文件。

3. 生成并部署：

```bash
hexo clean
hexo generate
hexo deploy
```

## 增强功能

您可以为博客添加更多功能：

1. **评论系统**：集成Valine、Gitalk等评论服务
2. **搜索功能**：添加本地搜索插件
3. **统计分析**：集成Google Analytics或百度统计
4. **RSS订阅**：生成RSS源供读者订阅

## 总结

使用Hexo和GitHub Pages搭建个人博客是一个简单、免费且高效的方式。您只需要一些基本的命令行操作，就能拥有一个功能完善的个人技术博客。
